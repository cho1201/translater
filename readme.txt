===========================================================
    VALORANT JP TRANSLATOR (발로란트 아섭 전용 번역기)
===========================================================

본 프로그램은 발로란트 일본 서버 플레이어들을 위한 
실시간 채팅 번역 및 의사소통 보조 도구입니다.

[ 주요 기능 ]

1. JP -> KR (듣기 및 읽기 모드)
   - 히라가나/한자 입력: 일본어 채팅을 복사해서 넣으면 즉시 한국어 해석 제공.
   - 로마자(영어) 입력 지원: 'gomennasai', 'otukare' 처럼 영어 발음으로 입력하면 
     자동으로 일본어로 변환 후 한국어 뜻을 출력.
   - 이미지 번역(OCR): 게임 화면 스크린샷을 업로드하면 이미지 속 일본어 한자나 
     가타카나를 자동으로 인식하여 번역.

2. KR -> JP (말하기 모드)
   - 단어 조합 패널: 위치(A사이트, 미드), 전략(러쉬, 세이브), 매너(나이스, 미안) 등 
     자주 쓰는 용어를 버튼 클릭만으로 조합하여 입력 가능.
   - 발음 가이드 제공: 번역된 일본어 문장과 함께 어떻게 읽는지 '히라가나 발음'을 
     별도로 표시해줌.
   - 원클릭 복사: 번역된 결과를 버튼 하나로 복사하여 게임 채팅창에 즉시 붙여넣기 가능.

3. 사용자 편의성
   - 발로란트 테마 UI: 게임 분위기에 맞춘 다크 모드 및 레드 포인트 디자인.
   - 단축키 지원: Ctrl + Enter 키로 즉시 번역 실행.
   - 배포 최적화: Render, Heroku 등 웹 서비스 배포에 최적화된 코드 구조.

[ 설치 및 실행 방법 ]

1. 필수 라이브러리 설치:
   pip install flask deep-translator pykakasi easyocr opencv-python-headless gunicorn

2. 프로그램 실행:
   python app.py

3. 웹 접속:
   브라우저 주소창에 http://127.0.0.1:5000 입력

[ 파일 구조 ]

- app.py            : 메인 서버 로직 (번역, OCR, 변환 처리)
- index.html        : 사용자 화면 디자인 및 프론트엔드 로직
- requirements.txt  : 설치에 필요한 라이브러리 목록
- README.txt        : 본 프로그램 설명서

[ 참고 사항 ]

- 이미지 번역(OCR) 기능은 첫 실행 시 일본어 인식 모델을 다운로드하느라 
  잠시 로딩이 발생할 수 있습니다.
- 영문 로마자 입력 시 오타가 있으면 변환이 정확하지 않을 수 있으니 
  정확한 발음(예: arigato, gomennasai)으로 입력해 주세요.
